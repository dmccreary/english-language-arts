# Pip the Bookworm — AI Image Prompts

Self-contained prompts for generating each of Pip's seven poses. Each
prompt embeds the full base character description so it can be used
independently. Generate at 1024x1024 PNG with a fully transparent
background, then run `trim-padding-from-image.py` on each file.

The base description below is the **source of truth** and must match
[`character-sheet.md`](character-sheet.md). If you edit one, edit both.

---

## Base Character Description

> A modern flat vector cartoon illustration of Pip the Bookworm, a friendly
> pedagogical mascot for a US high school English/Language Arts textbook.
> Pip is a small, plump segmented cartoon worm with an indigo (#3f51b5)
> body and subtle orange (#ff9800) accents. Pip wears small round black
> glasses — no other clothing, hats, scarves, or props. Pip has big
> friendly eyes and a gentle closed-mouth smile. The character is small
> and compact, suitable for icon-sized display (~90px square).
> Style: modern flat vector, clean lines, fully transparent background,
> suitable for embedding in educational content. No text in image.

## After Generation

1. Save each PNG into `docs/img/mascot/` with the exact filename above.
2. Trim transparent padding so each pose displays full-size inside its
   admonition box:

   ```bash
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/neutral.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/welcome.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/thinking.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/tip.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/warning.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/encouraging.png
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/celebration.png
   ```

3. Visit `/learning-graph/mascot-test/` while running `mkdocs serve` to
   preview every pose inside its admonition style.
