# Media Literacy and Cognitive Bias

## Summary

This chapter pairs two deeply connected critical-reading frameworks: media literacy and cognitive bias. The media literacy section develops the skills to evaluate sources, detect misinformation and propaganda, and navigate social media with skepticism — from source evaluation and fact-checking to recognizing emotional manipulation. The cognitive bias section surveys eighteen documented biases — from confirmation bias and the availability heuristic to motivated reasoning and stereotyping — that shape how readers and writers perceive information, interpret texts, and construct arguments. Together, these frameworks build the critical self-awareness required for college-level inquiry.
